쇼핑몰 숏폼 콘텐츠를 위젯 형태로 삽입하고, 상품과 연동하여 전환율을 높이는 영상 커머스 플랫폼

#### 주요 기능

- 쇼핑몰에 숏폼 콘텐츠를 삽입하여 상품과 직접 연동 가능
- **플로팅 UI, 3D 큐브 전환 위젯 등 다양한 시청 방식 지원**
- 영상 시청 로그 수집 및 **전환률 기반 분석 대시보드 제공**
- 시청 시간, 상품 클릭률 등 실제 행동 데이터 기반 인사이트 제공

#### 담당 역할 / 기여도

- **숏폼 영상 전용 위젯 구조 설계 및 신규 위젯 2종 개발**

  - 기존 Live24에 통합되어 있던 구조에서 Shorts24로 분리, 번들 사이즈 약 40% 감소
  - `Swiper.js`를 활용해 **인스타그램 스토리 스타일의 3D 큐브 전환 위젯** 구현
  - 플로팅 형태 및 스토리형 UI 등 다양한 형태의 위젯 제공

- **영상 시청 로그 수집 및 전환 분석 기능 구현**
  - Highcharts를 사용해 시청 시간, 상품 클릭률, 유입경로 등 분석 차트 구성
  - 데이터를 통해 쇼핑몰 운영자가 **영상 콘텐츠의 효과를 직접 판단할 수 있도록 시각화**
