#### MA(Media Analytics)

플레이어의 실시간 세션 로그를 수집·시각화하여 사용자 경험과 오류 상황을 정밀 분석하는 통합 분석 시스템

🔗 [Media Analytics 사용 가이드](https://guide.ncloud-docs.com/docs/ma-screen)

#### 주요 기능

- **VPE 연동 실시간 세션 로그 수집 + 시각화**
- **Highcharts 기반 인터랙션 시각화 UI** (툴팁, 확대 등)
- **전역 공통 필터 구조**로 페이지 간 조건 유지 및 비교 지원
- **에러 발생 시 알림 전송**, 장애 인지 속도 개선
- **i18n 적용 (한/영/일)**, 글로벌 서비스 대응 완료

---

#### 담당 역할 / 기여도

- **Nuxt 기반 전체 프론트 구조 설계 및 구현 총괄**
- **Highcharts 시각화 컴포넌트 개발** (인터랙션 포함)
- **전역 필터 시스템 설계 및 구현** (`useState`)
- **2차 인증 보안 기능 개발** (OTP + 이메일)
- **i18n 적용 및 언어 전환 UI + 테스트 대응**
