#### 담당 역할 / 기여도
- 반복되는 차트 구현 비용을 줄이기 위해 Highcharts를 공통 컴포넌트로 추상화하여, 대시보드, 리포트 전반에
서 재사용 가능한 구조 설계
- 페이지별로 분리되어 있던 필터 조건을 글로벌로 통합하여 분석 흐름의 일관성 개선
- OTP + 이메일 기반 2단계 인증 도입으로 CSAP 인증 요구사항 충족
- i18n 구조 설계를 통해 한국어, 영어, 일본어 다국어 지원 및 브라우저 언어 자동 감지 기능 구현
