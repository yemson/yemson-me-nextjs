라이브 방송 시작 시 사용자에게 푸시 알림을 전송하는, FCM 기반의 PWA 알림 시스템

#### 주요 기능

- **Next.js 기반 PWA 구성**으로 브라우저에 설치 가능한 모바일 웹 앱 제공
- 클라이언트에서 FCM 토큰 생성 후 서버에 전송 및 저장
- 라이브 방송 시작 시 저장된 토큰 대상 푸시 알림 발송
- **브라우저 설치 + 백그라운드 알림 지원**으로 실제 쇼핑몰 이용자에게 실시간 안내 가능

#### 담당 역할 / 기여도

- 전체 PWA 앱 구성 및 서비스 워커 설정
- 클라이언트에서 FCM 토큰 발급 후 서버로 전송하는 로직 구현
- Firebase 메시징 API를 활용한 푸시 알림 테스트 및 서버 연동
- 백엔드 개발자와 협업하여 **토큰 저장 → 알림 발송 플로우 전체 설계 및 테스트 주도**
