#### 주요 기능

- **Next.js 기반 PWA 구성**으로 브라우저 설치형 모바일 웹 앱 제공
- **FCM 토큰 생성** 및 서버 연동으로 사용자 대상 푸시 알림 준비
- 라이브 방송 시작 시 저장된 토큰 대상 **실시간 푸시** 발송
- 브라우저 설치 + 백그라운드 알림 지원으로 실사용자 **접근성 강화**

---

#### 담당 역할 / 기여도

- 전체 PWA 앱 구조 설계 및 서비스 워커 설정
- 클라이언트에서 FCM 토큰 발급 후 서버 전송 로직 구현
- Firebase Messaging API 기반 푸시 알림 테스트 및 서버 연동
- 백엔드 개발자와 협업하여 **토큰 저장 → 알림 발송 전체 플로우 설계 및 테스트 주도**
