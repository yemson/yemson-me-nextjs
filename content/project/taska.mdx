#### 담당 역할 / 기여도

- Firebase Auth, Firestore 기반으로 인증 및 태스크 관리 구조 설계
- 프로젝트, 버킷, 태스크 단위로 작업을 분리하는 정보 구조 설계
- Zustand 기반 전역 상태 관리로 화면 간 데이터 흐름 단순화
- 실데이터 기반 대시보드 차트 구성으로 작업 현황 시각화
- 핵심 기능 중심으로 제품 구조를 설계해 복잡도 최소화
