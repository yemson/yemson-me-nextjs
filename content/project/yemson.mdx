#### 담당 역할 / 기여도

- Next.js App Router와 MDX 기반으로 소개, 프로젝트, 블로그를 파일 중심으로 관리할 수 있는 구조 설계
- 프로젝트와 글 콘텐츠를 코드와 함께 버전 관리해 업데이트 비용을 줄이고 유지보수 흐름 단순화
- 정적 생성 중심으로 페이지를 구성하고 가벼운 인터랙션만 더해 빠른 로딩 경험과 읽기 흐름 유지
- 개인 브랜딩, 작업 아카이빙, 기술 글 정리를 하나의 사이트 안에서 자연스럽게 연결되도록 정보 구조 구성
