#### 담당 역할 / 기여도

- HLS 기반 영상 데이터를 활용한 드래그형 하이라이트 편집 UI(트리머) 구현 및 GSAP 기반 인터랙션 적용
- 숏폼 콘텐츠로의 빠른 전환 흐름을 구축하여 영상 시청 리텐션 약 10% 개선
- 서비스 전반에 분산되어 있던 알림/모달 구조를 공통 컴포넌트로 통합해 코드 중복 약 60% 감소
- Pinia 기반 전역 상태 관리 구조를 설계해 재사용성과 확장성 향상, 신규 기능 대응 속도 개선
